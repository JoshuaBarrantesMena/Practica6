¿Que es el Frontend?

El frontend es la parte de una aplicación o sitio web con la que los usuarios interactúan directamente. Se encarga de la interfaz de usuario (UI) y la experiencia de usuario (UX), asegurando que el diseño sea intuitivo, accesible y funcional en distintos dispositivos y navegadores.

Fundamentos del Frontend

El desarrollo frontend se basa en tres tecnologías principales:


  1. HTML (HyperText Markup Language)
  2. CSS (Cascading Style Sheets)
  3. JavaScript (JS)
